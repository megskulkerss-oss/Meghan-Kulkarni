# Meghan Kulkarni — Video Portfolio

Dark-mode video portfolio with YouTube embeds, filtering, and modal playback.

## Run locally
```bash
npm install
npm run dev
```

## Deploy (Option A — Vercel)
1. Create a free Vercel account (if you don’t have one).
2. Create a new project and import this folder (drag & drop upload in Vercel works, or push to GitHub and import).
3. Deploy.

To edit projects, open `src/data/projects.ts`.
To edit About/Contact, open `src/app/page.tsx`.
